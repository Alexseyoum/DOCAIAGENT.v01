üì° API Endpoints

Authentication

POST /api/v1/auth/register

Register a new user and get API key

POST /api/v1/auth/login

Login existing user

Document Management

POST /api/v1/documents/upload

Upload a document (requires API key)

POST /api/v1/documents/:id/process

Extract text from uploaded document

GET /api/v1/documents

List all your documents

DELETE /api/v1/documents/:id

Delete a document

Generate AI summary from document

POST /api/v1/generate/quiz

Generate quiz questions

POST /api/v1/generate/flashcards

Generate study flashcards

üíª Example Usage

1. Register & Get API Key

curl -X POST https://docaiagent-v01.onrender.com/api/v1/auth/register \ -H "Content-Type: application/json" \ -d '{ "email": "your@email.com", "password": "YourPassword123", "name": "Your Name" }' # Response includes your token (API key): # { # "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." # }

2. Upload Document

curl -X POST https://docaiagent-v01.onrender.com/api/v1/documents/upload \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -F "file=@document.pdf"

3. Process Document

curl -X POST https://docaiagent-v01.onrender.com/api/v1/documents/DOC_ID/process \ -H "Authorization: Bearer YOUR_TOKEN_HERE"

4. Generate Summary

curl -X POST https://docaiagent-v01.onrender.com/api/v1/generate/summary \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "documentId": "DOC_ID", "summaryType": "brief" }'
